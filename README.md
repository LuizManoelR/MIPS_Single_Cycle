# MIPS_Single_Cycle
Projeto acadêmico de Arquitetura de Computadores com implementação, em Verilog, de um modelo arquitetural MIPS de ciclo único baseado em subconjunto da ISA. Inclui datapath, unidade de controle e multiplexadores, além de multiplicação iterativa por soma e deslocamento com controle da atualização do PC. Simulação com Icarus Verilog e GTKWave.
